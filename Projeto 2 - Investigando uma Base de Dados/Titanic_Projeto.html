Investigando uma base de dados - Titanic¶

O RMS Titanic foi um navio de passageiros britanico que colidiu com um iceberg em 14 de abril de 1912 e naufragou durante a madrugada do dia seguinte.

O conjunto de dados que ser explorado nesse projeto tem como objetivo criar e responder algumas perguntas referentes as causas que influenciaram na chance de sobrevivencia dos passageiros.

O conjunto de dados possui informaes de 891 dos 2224 passageiros que se encontravam no navio no momento do naufrgio.

Primeiramente precisamos identificar os dados que existem no arquivo csv: Descrio das variaveis: PassengerId - Indicador unico dos passageiros na tabela. Survival - Indica se o passageiro sobreviveu. (0 = No; 1 = Sim) Pclass - Indica a classe do passageiro. (1 = 1 Classe; 2 = 2 Classe; 3 = 3 Classe) Name - Indica o nome do passageiro. Sex - Indica o genero do passageiro. Age - Indica a idade do passageiro. Sibsp - Indica a quantidade de parentes do passageiro a bordo do Titanic com os seguintes graus de parentesco: (Irmo, Irm, meio-irmo, meia-irm, marido e esposa). Parch - Indica a quantidade de parentes do passageiro a bordo do Titanic com os seguintes graus de parentesco: (Filhos, filhas, enteados, enteadas, pai e me). Ticket - Indica o nmero do Ticket do passageiro. Dare - Indica o custo da passagem do passageiro. Cabin - Indica a cabine na qual o passageiro se encontrava. Embarked - Indica o local de embarque do passageiro. (C = Cherbourg; Q = Queenstown; S = Southampton) Fonte: https://www.kaggle.com/c/titanic/dataVamos responder trs perguntas ao longo do projeto, so elas: 1 - Gnero foi um fator determinante na deciso de quem deveria sobreviver no naufragio do Titanic? 2 - Idade foi um fator determinante na deciso de quem deveria sobreviver no naufrgio do Titanic? 3 - Classe social foi um fator determinante no momento de se decidir quem teria maiores chances de sobrevivncia?

def barGraph(df, h):
    """Funo para gerar grficos de barras"""
    sns.set(style="white")

    #Definimos aqui o range de valores que seram exibidos no grafico (0 - Para no sobreviventes / 1 - Para sobreviventes)
    survived = np.arange(0, 2)

    #Criamos o grfico separando os sobreviventes pelas caracteristicas que desejamos ter no grfico
    g = sns.factorplot(x="Survived", data=df, hue=h,kind="count",
                       palette="BuPu", size=5, aspect=1, order=survived)

    #Retorna o grfico cos valores no eixo X sendo exibidos de 1 em 1
    return g.set_xticklabels(step=1)

def kdeGraph(df,h):
    """Funo para criar grficos de linhas por idade"""
    sns.set(style="white")

    #Definimos o grfico separando os sobreviventes pelas caracteristicas que desejamos no grfico
    graph = sns.FacetGrid(df,hue=h,aspect=3)

    #Define a diviso dos passageiros atravs da idade no eixo x
    graph.map(sns.kdeplot,'Age',shade='True')

    #Determina a extenso do eixo x no grfico, para ir dos passageiros de menor idade, at o mais velho
    graph.set(xlim=(0,df['Age'].max()))
    graph.add_legend()
    return graph

Existe uma chance alta de a classe social ter influenciado nas chances de sobrevivncia dos passageiros, quando dividimos o conjunto de dados por gnero essa relao ficou mais clara, principalmente quando analisamos a taxa de bitos da terceira classe das passageiras do sexo feminino, e a taxa de sobrevivncia da primeira classe dos passageiros do sexo masculino.
