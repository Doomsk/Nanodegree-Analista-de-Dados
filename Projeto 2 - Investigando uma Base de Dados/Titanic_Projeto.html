Investigando uma base de dados - Titanic¶

O RMS Titanic foi um navio de passageiros britanico que colidiu com um iceberg em 14 de abril de 1912 e naufragou durante a madrugada do dia seguinte.

O conjunto de dados que ser explorado nesse projeto tem como objetivo criar e responder algumas perguntas referentes as causas que influenciaram na chance de sobrevivencia dos passageiros.

O conjunto de dados possui informaes de 891 dos 2224 passageiros que se encontravam no navio no momento do naufrgio.

Primeiramente precisamos identificar os dados que existem no arquivo csv: Descrio das variaveis: PassengerId - Indicador unico dos passageiros na tabela. Survival - Indica se o passageiro sobreviveu. (0 = No; 1 = Sim) Pclass - Indica a classe do passageiro. (1 = 1 Classe; 2 = 2 Classe; 3 = 3 Classe) Name - Indica o nome do passageiro. Sex - Indica o genero do passageiro. Age - Indica a idade do passageiro. Sibsp - Indica a quantidade de parentes do passageiro a bordo do Titanic com os seguintes graus de parentesco: (Irmo, Irm, meio-irmo, meia-irm, marido e esposa). Parch - Indica a quantidade de parentes do passageiro a bordo do Titanic com os seguintes graus de parentesco: (Filhos, filhas, enteados, enteadas, pai e me). Ticket - Indica o nmero do Ticket do passageiro. Dare - Indica o custo da passagem do passageiro. Cabin - Indica a cabine na qual o passageiro se encontrava. Embarked - Indica o local de embarque do passageiro. (C = Cherbourg; Q = Queenstown; S = Southampton) Fonte: https://www.kaggle.com/c/titanic/data

#Funes que sero usadas ao longo da analise
#Funo para retornar uma lista de passageiros de um determinado gnero
def genderDef(df, gender):
    return df[df.Sex==gender]
#Funo para remover colunas na hora da exibio de dados e grficos
def removeColumns(df):
    return df.ix[:, ['Survived']]
#Funo para retornar quantidade de sobreviventes ou de bitos de um determinado grupo
def survival(df,condiction):
    return df[df.Survived==condiction].count()

2 - Idade foi um fator determinante na decisão de quem deveria sobreviver no naufrágio do Titanic?¶

Para responder essa pergunta podemos abordar de duas formas, uma delas observando a quantidade total de crianas a bordo do navio, no entando isso gera dois problemas: 1 - Ns no saberemos se o fator que impactou na escolha de resgate de algumas das crianas foi sua idade ou gnero. 2 - Ns no possuimos a idade de todos os tripulantes a bordo. Para resolver o primeiro problema, iremos analisar inicialmente apenas a idade dos passageiros do sexo masculino, assim saberemos se a idade dentro do grupo masculino foi de fato um fator importante na hora de escolher quem seria salvo. Quanto ao segundo problema, se considerarmos apenas o grupo de sexo masculino podemos resolv-lo parcialmente. Ao observar os nomes dos passageiros, notamos que eles possuem um pronome de tratamento no campo. (Ex: Mr. Miss). Dentre os titlos existe o ttulo 'Master' que de acordo com o livro "Honor & Respect - The Official Guide to Names, Titles, and Forms of Address" a forma utilizada para se descrever crianas jovens demais (at 12 anos) para serem chamadas de 'Mister'. Seguindo esse ponto, vamos considerar apenas os passageiros com at 12 anos como crianas, uma vez que na poca era dificil determinar quem era considerado uma criana, a exemplo disso, houveram crianas de at 14 anos que participaram da primeira guerra (iniciada dois anos depois do naufrgio), o que demonstra que a percepo de algum como criana era algo bem diferente do que a percepo atual de menores de 18 anos. Fontes: http://www.formsofaddress.info/Children.html http://www.bbc.com/news/magazine-29934965
